Quick Commerce Customer Experience Analytics Dashboard

An interactive Power BI dashboard designed to analyze customer experience across leading quick-commerce platforms including Blinkit, Zepto, Swiggy Instamart, and JioMart. The dashboard transforms customer review data into actionable insights through KPI tracking, sentiment analysis, delivery performance monitoring, and service quality evaluation.

 Project Highlights

* Analyzed 5,000 customer review records
* Built interactive KPI cards and slicers
* Created DAX measures for customer experience metrics
* Developed delivery performance and sentiment analysis visualizations
* Designed an executive-level dashboard using Power BI

 Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization
* Business Intelligence
